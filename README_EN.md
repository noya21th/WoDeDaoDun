## 🛡️ Shield — skill-audit

**For managers.** Scan any Skill document in 30 seconds. Five-dimension scoring: Specificity · Causality · Contextuality · Experience · Personality.

| Document type | Typical score | Verdict |
|---------------|:---:|---------|
| Honestly written | 75-90 (A) | "Excellent, adopt directly" |
| Hollowed-out shell | 10-40 (D/F) | "Severely hollow, recommend discussion" |
| Knowledge-encrypted | 65-80 (B) | "Solid, some areas could be more detailed" |

## ⚔️ Sword — skill-encrypt

**For professionals.** Deliver an 80%-solid great document, keep the 20% critical judgment calls as your key.

Four strategies: Boundary Encryption · Dependency Anchoring · Process Compression · Condition Encapsulation

Four unique capabilities:
- **Situation Awareness** — Different strategies for different career situations
- **Reverse Verification** — Simulates scenarios to prove the doc actually works
- **Value Visualization** — Generates an irreplaceability report with scores
- **Gradual Decryption** — Phase-based key release for handovers
