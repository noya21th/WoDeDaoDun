企業が社員の経験をAI Skillに蒸留している。社員は空の文書で対抗する。<br>
管理者は誰がごまかしているか分からない。社員も空の文書がバレやすいと知っている。<br>

**刀盾はより高度な解決策。**<br>
盾は管理者が30秒で空洞な文書を見抜くツール。<br>
刀はプロフェッショナルが優れた文書を提出しつつ、不可欠な存在であり続けるツール。<br>

**刀で作成された文書は、盾の監査を自然に通過する。**<br>
不正ではなく、本質的に優れた文書だから。

## 🛡️ 盾 — skill-audit

管理者向け。Skill文書を30秒でスキャンし、五次元で評価：具体性・因果性・文脈性・経験性・人格性。

→ [盾の詳細](shield/README.md)

## ⚔️ 刀 — skill-encrypt

プロフェッショナル向け。80%は充実した高品質文書を提出し、20%の重要な判断力をキーとして保持。

→ [刀の詳細](sword/README.md)
